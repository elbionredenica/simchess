# SimChess - Simultaneous Chess

A real-time multiplayer chess game where both players submit their moves simultaneously.

## How to Run

### Prerequisites
- Python 3.7+
- pip

### Create a virtual environment

```bash
python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

### Installation

1. Install dependencies:
```bash
pip install -r requirements.txt
```

### Running Locally

```bash
flask run
```

or

```bash
python app.py
```
